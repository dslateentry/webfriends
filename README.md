# WebFRIENDS

Hub social privado Y2K para grupo de amigos. Estética Neocities/Gyaru/Harajuku.

## 🚀 Cómo publicar (IMPORTANTE)

Los reproductores de YouTube y Spotify **requieren HTTP**. Hay 2 opciones:

### Opción A: Hostear gratis (Recomendado)
1. Creá cuenta en [GitHub](https://github.com)
2. Subí `index.html` a un repo
3. Activá **GitHub Pages**: Settings > Pages > Source: main > Save
4. Tu URL: `https://TUUSUARIO.github.io/webfriends`
5. Compartí ese link. YouTube y Spotify funcionan incrustados.

**Alternativas gratis:** [Netlify](https://app.netlify.com/drop) (arrastrá index.html) o [Vercel](https://vercel.com)

### Opción B: Abrir en local (limitado)
- Doble-click en `ABRIR.bat` inicia un mini-servidor invisible
- YouTube/Spotify funcionan, pero solo en esa PC
- Para compartir, usá la Opción A

## 📁 Archivos
- `index.html` — Toda la app (SPA + 8 vistas + 4 popups)
- `WebFRIENDS.ps1` — Servidor auto-contenido
- `ABRIR.bat` — Launcher simplificado
- `schema.sql` — Base de datos Supabase
